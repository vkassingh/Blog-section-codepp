Learn Git

Git is a must learn version control system.

What is a versin control system?

This can be understood with a simple example. Different people make different changes in the codebase of an app at different time. So to keep a track of the changes, their contributors, their time etc, we use git. You must of heard about version 1, version 2 and so on of an app. So likewise git is a version control system. It controls the versions of an application. It gives us the info about the changes made to a github repository that what changes are made, who made the changes, at what time and so on. It is an amazing tool ; hope you would like it too.
